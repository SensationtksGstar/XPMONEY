# Mascot Prompt Pack — Leonardo.ai (gratuito)

Este guia gera **11 renders 3D-style consistentes** das mascotes XP Money: 6 evoluções do Voltix (macho, dragão-trovão azul) e 5 da Penny (fêmea, gata-anjo creme).

## Setup uma vez no Leonardo

1. Cria conta em **https://app.leonardo.ai** (tem 150 tokens/dia grátis)
2. Modelo: **Leonardo Phoenix 1.0** (melhor para personagens 3D-style gratuito)
3. Preset Style: **3D Animation** ou **Stylized Illustration**
4. Aspect Ratio: **1:1 (1024 × 1024)**
5. Activa **Alchemy** e **PhotoReal Off** (queremos estilo Pixar/3D, não fotorrealista)
6. **Negative prompt global** (cola em todos):
   ```
   2D, flat, sketch, pencil, watercolor, anime lineart, low quality, blurry, ugly,
   bad anatomy, extra limbs, extra fingers, text, watermark, signature, cropped,
   out of frame, dark background, noisy background, cluttered, messy
   ```

## Regra chave para consistência

Usa sempre o mesmo `seed` dentro de cada género, e o mesmo *character anchor* em todos os prompts. Gera o EVO 1 primeiro, copia o seed, fixa-o nos restantes.

## Depois de gerares

1. Cada imagem gerada no Leonardo → **Universal Upscaler 2x** (mais detalhe) → **Canvas Editor** → **AI Background Remover**
2. Descarrega o PNG com fundo transparente
3. Abre **http://localhost:3000/admin/mascot-upload** no teu PC
4. Arrasta cada PNG para o slot correcto (Volt → EVO 1 do Voltix, Pennini → EVO 1 da Penny, etc.)
5. O servidor recorta, centra, redimensiona para 512×512 e grava como WebP em `public/mascot/<gender>/<n>.webp`
6. Vai a **/voltix** ou à dashboard — as imagens reais substituem os SVGs automaticamente
7. `git add public/mascot && git commit -m "assets: mascotes 3D-style" && git push` → deploy

## Dicas

- Se uma imagem sair off-model, usa o **Image Guidance** no Leonardo com a EVO 1 como referência (Character Reference, strength 0.4)
- Para manter a paleta perfeita, adiciona no final do prompt: `exact color palette: #1e3a8a sapphire blue, #06b6d4 cyan, #ef4444 ruby, #fef3c7 ivory, #eab308 yellow`
- Se o fundo não ficar totalmente transparente, o pipeline de sharp em `processOne()` ignora pixels com alpha < 10 no trim, mas idealmente o removedor do Leonardo dá alpha 0 nos cantos
